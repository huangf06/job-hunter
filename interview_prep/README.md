# Interview Prep Hub

## Workflow

Standardized 7-phase interview prep process documented in [`memory/interview_prep_workflow.md`](../memory/interview_prep_workflow.md).

Reference case study: [`20260225_Source_ag_Data_Engineer/00_process_log.md`](20260225_Source_ag_Data_Engineer/00_process_log.md)

### Canonical File Structure
```
00_process_log.md               ← How we prepared (meta-document)
01_job_description.md           ← Full JD + metadata
02_ai_analysis.md               ← AI scores, reasoning, recommendation
03_submitted_resume.md          ← Resume content (+ CL if applicable)
04_company_deep_dive.md         ← Company research dossier
05_interview_strategy.md        ← Narratives, Q&A, interviewer profile, team intel
06_quick_reference.md           ← One-page cheat sheet for during the call
07_transcript_and_education.md  ← Academic background
08_take_home_prep.md            ← (if multi-round) Assignment intel + prep strategy
09_post_interview_notes.md      ← (after interview) What happened, lessons learned
Fei_Huang_Resume.pdf            ← Copy of submitted resume
```
