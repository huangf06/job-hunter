## Key Preparation Tips (from FareHarbor)

### For Technical Screen (Tomorrow):

1. **Communication is Critical**
   - Think out loud
   - Explain your reasoning before coding
   - Ask clarifying questions
   - Discuss trade-offs

2. **Code Quality Matters**
   - Write clean, readable code
   - Use meaningful variable names
   - Handle edge cases
   - Test your solution

3. **Problem-Solving Approach**
   - Start with brute force, then optimize
   - Consider time/space complexity
   - Discuss alternative approaches
   - Be honest about what you don't know

4. **Python Fundamentals**
   - Data structures (lists, dicts, sets)
   - List comprehensions
   - Functions and classes
   - Common libraries (numpy, pandas)

5. **ML Concepts**
   - Supervised vs unsupervised learning
   - Model evaluation metrics
   - Overfitting/underfitting
   - Feature engineering basics

## Common Pitfalls to Avoid

❌ **Don't**:
- Jump straight to coding without understanding the problem
- Code in silence for long periods
- Give up when stuck
- Dismiss hints or feedback
- Claim to know something you don't

✅ **Do**:
- Clarify requirements upfront
- Communicate your thought process
- Ask for hints if genuinely stuck
- Be receptive to feedback
- Admit knowledge gaps honestly
