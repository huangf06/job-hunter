# 量化交易知识速查表

## 📚 基础概念

### 波动率 (Volatility)
| 概念 | 定义 |
|------|------|
| **历史波动率** | 基于过去价格变动的统计度量 |
| **隐含波动率 (IV)** | 从期权价格反推的市场预期波动率 |
| **波动率微笑** | 不同执行价期权的隐含波动率曲线 |
| **波动率期限结构** | 不同到期日期权的隐含波动率关系 |

### 期权希腊字母 (Greeks)
| 希腊字母 | 含义 | 公式符号 |
|----------|------|----------|
| **Delta (Δ)** | 期权价格对标的资产价格的敏感度 | ∂V/∂S |
| **Gamma (Γ)** | Delta 对标的资产价格的二阶导数 | ∂²V/∂S² |
| **Theta (Θ)** | 时间衰减 (每天损失) | ∂V/∂t |
| **Vega (V)** | 对隐含波动率的敏感度 | ∂V/∂σ |
| **Rho (ρ)** | 对利率的敏感度 | ∂V/∂r |

### 主要期权策略
| 策略 | 描述 | 适用场景 |
|------|------|----------|
| **Straddle** | 同时买入同一执行价的看涨和看跌期权 | 预期大波动，不确定方向 |
| **Strangle** | 买入不同执行价的看涨和看跌期权 | 比 straddle 成本低 |
| **Butterfly** | 组合多个期权限制风险和收益 | 预期低波动 |
| **Calendar Spread** | 不同到期日的期权组合 | 利用期限结构 |

## 🧮 核心公式

### Black-Scholes 模型
欧式看涨期权价格：
```
C = S₀N(d₁) - Ke^(-rT)N(d₂)

其中：
d₁ = [ln(S₀/K) + (r + σ²/2)T] / (σ√T)
d₂ = d₁ - σ√T

S₀ = 当前标的资产价格
K = 执行价格
T = 到期时间 (年)
r = 无风险利率
σ = 波动率
N() = 标准正态分布累积函数
```

### 波动率计算
对数收益率的样本标准差：
```
σ = √(252) × √[Σ(rᵢ - r̄)² / (n-1)]

其中 rᵢ = ln(Pᵢ / Pᵢ₋₁)
```

### Sharpe Ratio
```
Sharpe = (Rₚ - Rբ) / σₚ

Rₚ = 策略收益
Rբ = 无风险利率
σₚ = 策略收益波动率
```

## 💻 机器学习在量化中的应用

### 常见应用
| 应用 | 方法 | 挑战 |
|------|------|------|
| **价格预测** | 回归、时间序列模型、LSTM | 低信噪比、非平稳性 |
| **波动率预测** | GARCH、机器学习集成 | 集群效应、长记忆性 |
| **执行优化** | 强化学习、最优控制 | 市场冲击、延迟 |
| **风险管理** | 聚类、异常检测 | 尾部风险、相关性突变 |
| **特征工程** | 技术分析指标、订单流特征 | 过拟合、多重比较问题 |

### 过拟合防范
1. **样本外测试**: 训练/验证/测试集分离
2. **交叉验证**: 时间序列交叉验证
3. **正则化**: L1/L2 正则化
4. **特征选择**: 避免数据挖掘偏差
5. **交易成本**: 始终纳入实际交易成本

## 🏛️ 市场微观结构

### 关键概念
| 概念 | 定义 |
|------|------|
| **买卖价差 (Bid-Ask Spread)** | 最佳买价和卖价之差 |
| **市场深度** | 订单簿中各价位的挂单量 |
| **订单流** | 买卖订单的到达过程 |
| **市场冲击** | 大额订单对价格的影响 |
| **延迟 (Latency)** | 从信号产生到执行的时间 |

### 订单类型
- **Market Order**: 立即以当前市场价格执行
- **Limit Order**: 指定价格执行
- **Stop Order**: 达到触发价格后变为市价单
- **Iceberg Order**: 只显示部分数量的大单

## 🔢 数学基础

### 随机过程
| 过程 | 描述 | 应用 |
|------|------|------|
| **布朗运动 (Wiener Process)** | 连续时间随机游走 | 资产价格建模 |
| **几何布朗运动** | dS = μSdt + σSdW | Black-Scholes 基础 |
| **Ornstein-Uhlenbeck** | 均值回归过程 | 利率、波动率建模 |
| **泊松过程** | 跳跃过程 | 跳跃扩散模型 |

### 关键统计概念
- **平稳性 (Stationarity)**: 时间序列的统计特性不随时间变化
- **自相关 (Autocorrelation)**: 序列与自身滞后值的相关性
- **协整 (Cointegration)**: 非平稳序列的线性组合可能是平稳的
- **异方差性 (Heteroskedasticity)**: 波动率随时间变化

## 🎯 回测要点

### 常见陷阱
| 陷阱 | 描述 | 解决方案 |
|------|------|----------|
| **前视偏差** | 使用未来信息 | 严格按时间顺序回测 |
| **生存偏差** | 只考虑现存资产 | 包含已退市/破产资产 |
| **过度优化** | 参数过度拟合历史 | 样本外测试、正则化 |
| **忽视成本** | 未考虑交易成本 | 包含佣金、滑点、冲击 |
| **数据窥探** | 多次测试同一假设 | Bonferroni 校正、更严格标准 |

### 回测框架要素
```python
# 基本结构
1. 数据获取与清洗
2. 信号生成 (基于历史数据)
3. 组合构建 (仓位管理)
4. 执行模拟 (考虑成本)
5. 绩效评估 (收益率、风险指标)
```

## 📊 绩效指标

### 收益指标
| 指标 | 计算 | 说明 |
|------|------|------|
| **总收益** | (最终价值/初始价值) - 1 | 绝对收益 |
| **年化收益** | (1 + 总收益)^(252/天数) - 1 | 标准化比较 |
| **超额收益 (Alpha)** | 策略收益 - 基准收益 | 主动管理能力 |

### 风险指标
| 指标 | 计算 | 说明 |
|------|------|------|
| **波动率** | 收益率标准差 × √252 | 总风险 |
| **最大回撤** | (峰值-谷值)/峰值 | 最坏情况损失 |
| **VaR** | 给定置信水平的最大损失 | 风险价值 |
| **下行波动率** | 负收益的标准差 | 只考虑下行风险 |

### 风险调整收益
| 指标 | 公式 | 特点 |
|------|------|------|
| **Sharpe Ratio** | (Rₚ-Rբ)/σₚ | 总风险调整 |
| **Sortino Ratio** | (Rₚ-Rբ)/σ₋ | 下行风险调整 |
| **Calmar Ratio** | 年化收益/最大回撤 | 极端风险调整 |
| **Information Ratio** | Alpha/Tracking Error | 相对基准 |

## 💡 面试常见问题速答

### Q: "What is the difference between historical and implied volatility?"
**A**: 
- 历史波动率：基于过去价格变动的实现波动率
- 隐含波动率：从期权市场价格反推的未来预期波动率
- 隐含波动率通常高于历史波动率（风险溢价）

### Q: "How do you detect overfitting in a trading strategy?"
**A**:
- 训练集表现远好于测试集
- 参数对微小变化过于敏感
- 策略过于复杂（参数过多）
- 在随机数据上也能产生"好"结果

### Q: "What is market microstructure?"
**A**:
- 研究市场参与者的行为和互动
- 包括订单簿动态、流动性、价格发现过程
- 对高频交易和最优执行至关重要

### Q: "Why might two assets with correlation 1 not be perfectly cointegrated?"
**A**:
- 相关性衡量线性关系的强度
- 协整衡量长期均衡关系
- 两个资产可以同时上涨/下跌（高相关）但没有回归均值的倾向（不协整）
