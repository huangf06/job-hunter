# Job Description — Senior MLOps / ML Engineer

**Company:** Elsevier BV (part of RELX Group)
**Title:** ML Engineer — Research Products (A&G and STMJ)
**Location:** Amsterdam, Netherlands (Hybrid)
**Reports to:** Manager, Data Science
**Job ID:** R105530-1
**URL:** https://relx.wd3.myworkdayjobs.com/en-US/relx/job/Senior-MLOps_R105530-1

## About the Role

Join the team that powers Elsevier's research platforms — Scopus/Scopus AI, ScienceDirect/ScienceDirect AI, and journal submission & peer review workflows. You will bridge Data Science and Engineering to turn experimental NLP/IR/GenAI models into secure, reliable, and scalable services. Systems operate over one of the world's largest scholarly corpora — AI-based features (GenAI, Agentic AI, RAG, etc.), search/ranking quality, and knowledge graph aware retrieval while enforcing content rights and editorial confidentiality.

## Key Responsibilities

### ML & LLM Engineering, Search and Recommendation Engines
- Automate and orchestrate ML workflows across AWS, Azure, Databricks, and foundation model APIs (OpenAI)
- Maintain and version model registries and artifact stores for reproducibility and governance
- Develop CI/CD for ML: automated data validation, model testing, deployment
- Implement ML Engineering solutions using SageMaker, MLflow, Azure ML
- End-to-end custom SageMaker pipelines for recommendation systems
- Design GAR+RAG systems (query interpretation, reflection, chunking, embeddings, hybrid retrieval, semantic search), manage prompt libraries, guardrails and structured output for LLMs (Bedrock/SageMaker)
- Design ML pipelines utilizing Elasticsearch/OpenSearch/Solr, vector DBs, and graph DBs
- Build evaluation pipelines: offline IR metrics (NDCG, MAP, MRR), LLM quality metrics (faithfulness, grounding), A/B testing
- Optimize infrastructure costs through monitoring, scaling, efficient resource utilization
- Stay current with GAI research, NLP, RAG; apply state-of-the-art

### Collaboration
- Partner with SMEs, Product Managers, Data Scientists and Responsible AI experts
- Collaborate with Operations Engineers for production infrastructure

## Requirements
- 4+ years in ML Engineering, MLOps platforms, shipping ML or search/GenAI systems to production
- Strong Python; Java/Scala a plus
- Statistical analysis, ML theory, NLP
- Hands-on cloud experience (AWS, Azure, GCP)
- Search/vector/graph technologies (Elasticsearch/OpenSearch/Solr/Neo4j)
- LLM evaluation experience
- Background with scholarly publishing workflows, bibliometrics, or citation graphs (nice-to-have)
- Data Science Life Cycle: feature engineering, model training, evaluation metrics
- ML frameworks: PyTorch, TensorFlow, PySpark
- Large-scale data processing (Spark)

## Key Technical Areas (for interview prep)
1. **MLOps/ML Engineering** — CI/CD for ML, model registries, SageMaker pipelines
2. **RAG/GenAI** — Retrieval-augmented generation, prompt engineering, guardrails, structured output
3. **Search/IR** — Elasticsearch, NDCG/MAP/MRR metrics, vector search, hybrid retrieval
4. **Cloud** — AWS (SageMaker, Bedrock), Azure ML, Databricks
5. **Data Processing** — PySpark, large-scale ETL
6. **Knowledge Graphs** — Neo4j, graph-aware retrieval
