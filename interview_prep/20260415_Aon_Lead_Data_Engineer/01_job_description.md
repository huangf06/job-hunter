## Role Summary

Lead Data Engineer at Aon, responsible for designing, building, and optimizing data infrastructure on Azure Databricks platform. Key mandate: consolidate from Oracle + Azure Synapse + AWS Databricks to a unified Azure Databricks environment.

## Key Requirements

### Must-Have
- 7+ years data engineering experience with cloud-based solutions
- Azure (compute/storage), ideally DP-203 or DP-700 certified
- Databricks (Unity Catalog, Delta Lake, DABs, Genie, Workflows)
- Python, PySpark
- T-SQL / PL-SQL
- Data Vault / Dimensional modeling
- Data governance frameworks
- CI/CD (Azure DevOps)
- Technical leadership of multi-functional teams

### Migration Mandate
Oracle + Azure Synapse + AWS Databricks → Azure Databricks (unified platform)

### Nice-to-Have
- Insurance/financial services domain experience
- Experience with large-scale data migrations
- Solution/Data Architect career path interest
