## Review Criteria

Please evaluate the resume from the perspective of:
1. **Recruiter 10-second scan** - Does it pass the initial filter?
2. **Hiring manager first read** - Does it create interest or skepticism?
3. **Interview risks** - What questions will this document trigger?

## Specific Questions

1. **Narrative clarity**
   - Is the core story coherent now?
   - Does it clearly position me as a Data Engineer?
   - Are there STILL any signals of career drift, quant drift, or academic drift?

2. **Credibility**
   - Are any claims still inflated, vague, or hard to defend in interview?
   - Are any lines likely to trigger skepticism?
   - Do the dates, titles, and certifications feel credible?

3. **Relevance for DE roles**
   - Which bullets STRONGLY support Data Engineering?
   - Which lines still feel too quant / too ML / too generic / too weak?
   - Is the project section helping or hurting?

4. **Gap / transition handling**
   - Does the current treatment of the 2019-2023 period work NOW?
   - Is it explained enough, or still risky?
   - Is it visually balanced and appropriately low-profile?

5. **Layout / scanability**
   - Does the page feel balanced?
   - Any sections too dense, too visually loud, or too weak?
   - Any line breaks, labels, or formatting choices that hurt readability?

## Output Format

1. **Overall judgment**
   - "Ready to send" or "Not ready to send"
   - 3-5 sentence summary

2. **High-severity findings** (if any)
   - Only items that would materially reduce interview chances
   - For each: explain why it is a problem and exactly how to fix it

3. **Medium-severity findings** (if any)
   - Important but non-fatal issues
   - Again: explain and propose specific fixes

4. **Best lines**
   - Quote the strongest bullets / phrases
   - Explain why they work

5. **Weakest lines**
   - Quote the weakest bullets / phrases
   - Rewrite them if possible

6. **Final decision**
   - If I send this now, what is the biggest residual risk?
   - If I can only fix 2 more things, what should I fix?
